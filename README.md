# DND Tavern Prototype

Прототип игры в стиле "таверна + выживание" на базе Electron.

## Структура

- `Game/` — исходный код проекта
- `Game/server/` — кооперативный сервер
- `Game/assets/` — игровые ресурсы

## Быстрый старт

1. Перейдите в папку проекта:

```bash
cd Game
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npm start
```

## Дополнительно

- Запуск кооперативного сервера:

```bash
npm run coop-server
```

- Сборка portable-версии:

```bash
npm run dist
```
