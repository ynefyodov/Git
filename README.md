# DND Tavern Prototype

Легковесный прототип top-down action game в стиле "таверна + выживание".
Сейчас основной сценарий распространения - web-версия (single-player), доступная через GitHub Pages.

## Визуал (Live Demo)

- Играть в браузере: [https://ynefyodov.github.io/Git/](https://ynefyodov.github.io/Git/)
- Если видишь `404`, подожди 1-5 минут после деплоя и обнови страницу.

## Суть проекта

- 2D арена с волнами врагов, прокачкой черт и переходом между локациями.
- Полный конструктор персонажа: раса -> класс -> ветка развития.
- 6 классов, 8 рас, активные умения и пассивные модификаторы билда.
- Быстрые игровые сессии для тестирования геймплея и баланса.
- Web-прототип без кооператива (удобно делиться с друзьями по ссылке).

## Новая концепт-версия (full-sheet)

- **Расы**: люди, эльфы, дварфы, гномы, орки, гоблины, демоны, нежить.
- **Классы**: воин, варвар, разбойник (мили), разбойник (рейндж), маг, прист.
- **Ветки**: по 4 ветки на класс (броня/дальность/урон/вампиризм или скорость/контроль по роли).
- **Активные скиллы**:
  - Воин — Щит
  - Варвар — Ярость
  - Разбойник мили — Ускользание
  - Разбойник рейндж — Скорость ветра
  - Маг — Цепь
  - Прист — Благодать

## Adventure Map Full Feature

- После боевых узлов появляется схематичная карта-перекресток с выбором:
  - `Лево` / `Прямо` / `Право` — переход в следующую боевую локацию;
  - `Отдых` — восстановление HP, баф `Отдохнувший`, пропуск XP на следующем боевом шаге.
- Уникальные локации:
  - `Лес` — редкие, но опасные враги + ловушки с телеграфом;
  - `Река` — высокоплотные волны мурлоков;
  - `Неизвестно (?)` — смешанные события и нестабильная сложность.
- Сложность растёт от перекрестка к перекрестку:
  - больше врагов,
  - выше здоровье и урон,
  - выше интенсивность опасностей.

## Как скачать и запустить игру

### Вариант 1: Запустить сразу в браузере (рекомендуется)

- Открой ссылку: `https://ynefyodov.github.io/Git/`
- Ничего устанавливать не нужно.
- Управление: движение `WASD/стрелки`, прыжок `Space`, активный скилл `E`.

### Вариант 2: Скачать архив с игрой

1. Скачай файл `share/DND-Tavern-Web-Prototype.zip` из репозитория.
2. Распакуй архив в любую папку.
3. Открой файл `index.html` в браузере.

## Запуск из исходников (для разработки)

1. Клонируй репозиторий.
2. Перейди в папку проекта:

```bash
cd Game
```

3. Установи зависимости:

```bash
npm install
```

4. Запусти desktop-версию (Electron):

```bash
npm start
```

## Полезные пути в репозитории

- `Game/` - код игры
- `Game/src/` - основная игровая логика
- `Game/assets/` - графика и игровые ресурсы
- `share/DND-Tavern-Web-Prototype.zip` - архив для быстрого распространения

## Автоматизация GitHub

- `/.github/workflows/deploy-pages.yml` - автопубликация web-версии на GitHub Pages
- `/.github/workflows/agent-ci-failure-issue.yml` - авто-issue при падении деплоя
- `/.github/workflows/agent-issue-autopick.yml` - автоподхват issue с меткой `agent:todo`
